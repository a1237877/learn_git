git is a  distributed 1  version control system
git is free software under the GPL
git has a mutable index stage.
git tracks changes.


git init 初始化该文件夹
cd .git 进入GIT隐藏文件
cd ../   退出隐藏文件
git add readme.txt 添加文件
git commit -m 确认添加
git status 查看git处于什么状态
git diff readme.txt 查看和上一次提交的文件的差别
git log 查看版本
